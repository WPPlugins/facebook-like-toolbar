== Description ==

Facebook like toolbar plugin display at the bottom or top of your page window/toolbar with custom message and facebook page like button. You can easily hide this window using close icon or just wait a second to automatically hide.

After installation go to "Settings -> Facebook like toolbar” and customize plugin options.

== Installation ==

Download and simply install plugin. There’s no extra configuration needed.

After installation go to "Settings -> Facebook like toolbar" to customize plugin options.
